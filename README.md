# Проект Mesto (работа с сервером)

Интерактивная страница с профилем пользователя и карточками мест. Данные загружаются и сохраняются через API.

## Ссылка на опубликованный проект

https://arch1k1.github.io/mesto-ad-site/

## Локальный запуск

Установка зависимостей:

```bash
npm install
```

Запуск dev-сервера (страница откроется в браузере автоматически):

```bash
npm run dev
```

Сборка в папку `dist`:

```bash
npm run build
```

Публикация собранного проекта на GitHub Pages:

```bash
npm run deploy
```
